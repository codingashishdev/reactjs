## üö¶ Getting Started

### Installation

To run any project:

1. Navigate to the project directory:
    ```bash
    cd [project-folder]
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Start the development server:
    ```bash
    npm run dev
    ```

4. Build for production:
    ```bash
    npm run build
    ```
